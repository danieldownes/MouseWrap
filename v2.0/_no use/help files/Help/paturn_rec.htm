Mouse Wrap 2

Pattern Movement Recognition
 

Movement Pattern Recognition allows actions to be triggered without clicking a mouse button.

The Movement Patterns recognised are:

Rapid Horizontal Movement:

 

 
The mouse is moved repeatedly from side to side 3-4 times within the set amount of time.

Rapid Vertical Movement:

The mouse is moved repeatedly up and down 3-4 times within the set amount of time.

To set the desired actions, or to change the time-out period right-click the Mouse Wrap tray icon and choose 'Options'.

Currently the available actions are: Cut, Copy, Paste, Forward and Backward navigation in Windows Explorer and Internet Explorer. Also an individual Movement Pattern Recognition trigger can be disabled from the list.

Experiment with the time-out value to fit your personal preference. This value represents the amount of time a Movement Pattern is recognised. For example, if the mouse is moved repeatedly up and down 3-4 times within the set amount of time, then action triggered.


After making changes you should manually close and then start Mouse Wrap 2 so that changes are applied.
