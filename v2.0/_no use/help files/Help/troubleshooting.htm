Mouse Wrap 2

Troubleshooting
 

Mouse Wrap 2 does not load on start-up / log-on, even though that option is selected

Mouse Wrap 2 is a user independent software. This means that each user of a computer have their own settings. So if Mouse Wrap 2 was installed on the same computer but while user (A) was logged on, user (B) will not be able to use Mouse Wrap 2 until it is instilled while (B) is logged on, and thus, none of user (A)'s settings will be apparent.
Note, user (B)'s account might have to be temporally set up as an 'administrator' account in order to make the installation, depending on user (B)'s account privileges.


Mouse Wrap 2 doesn't allow the cursor to do beyond the primary display device in a multi display device environment

Mouse Wrap 2 currently does not support multi display device environments. Please contact Ex-D Software Development TM to suggest this for future releases; making a donation would dramatically improve the likelihood of your suggestion being implemented, although is not necessary.


While playing certain games, Mouse Wrap 2 interferes with the controls

Generally, playing games with Mouse Wrap 2 enabled will not affect the game at all. On rarer games (particularly those that do not use DirectX or DirectInput) the controls can be affected. In this situation it is recommended that you disable Mouse Wrap by left clicking its icon in the taskbar.

Additionally, the pattern recognition features of Mouse Wrap can also affect game play (even for those games that do utilise DirectX), in this case the pattern recognition feature should be disabled from the Options dialogue.


 

In most cases, un-installing, and then re-installing should eradicate most problems.
If problems persist, please contact Ex-D Software DevelopmentTM.
